# npe-image-library
An additional image library for graphics programming in Java.

**Under construction.**

##Features

- 100% Pure Java
- designed to run on multi-platform
- provides 2D and 3D graphics functions

##Support format

|Format|Read|Write|
|:-:|:-:|:-:|
|DDS|o|x|
|PSD|o|x|
|TGA|o|o|
